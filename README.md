# The Oregon Trail

## Getting Started

`git clone git@github.com:turingschool-examples/oregon-trail.git`  
`cd oregon-trail`  
`npm install`  

## Running the tests
`npm test [path to test file]`  

or, to autorun using nodemon  

`npm test:autorun [path to test file]`  

## Iterations

Suggested order of completion:  
1) Gear
2) Part
3) Settler
4) Wagon
5) Shop
